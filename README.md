# DaCaller — Ministry Outreach Dialer

An offline-first, mobile-friendly Progressive Web App for managing contacts and logging outreach calls.

## Features

- **PWA Ready** — Install on your mobile home screen
- **Dark Mode** — Auto-detects system preference, toggle anytime
- **Pull to Refresh** — Mobile-native gesture support
- **Offline-First** — All data stored locally in your browser
- **Smart Add** — Paste any text to auto-extract name & number
- **Bulk CSV Import** — Upload or paste CSV records
- **Deduplication** — Auto-detect and merge duplicate contacts
- **Call Logging** — Track Answered / Missed / Pending status
- **Backup & Restore** — Export/import your data as JSON

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
